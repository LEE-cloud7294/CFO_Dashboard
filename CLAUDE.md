## 1. 프로젝트 정의

**한 줄 정의:** 단순 CFO 대시보드가 아닌, 회사의 방향성을 결정하는 경영 의사결정 지원 도구.
인건비 비중이 높아 줄여야 하는지, 원재료가 상승하는지 등 모든 방향성을 판단할 수 있는 툴.
CEO / CFO 최고관리자 단 1인 전용.

**세 가지 역할:**
1. 과거 진단 — 숫자가 어떻게 변했나 (추세)
2. 현재 신호 — 지금 어디가 문제인가 (경보)
3. 방향 근거 — 이 추세면 다음 달 어떻게 되나 (판단 근거)

**사용자:** 경영자 단 1인. 차변/대변 화면 노출 금지.

**입력:** 위하고 월별 분개장 CSV.
- 보유 데이터: 2022년(4~12월) / 2023 / 2024 / 2025 전체
- 매달 1개월치 추가 업로드로 누적
- 2022년은 공장 매매 이벤트 있는 해 (자산 급변 정상)

**경영 데이터 내보내기:**

## 7. 데이터 정제 로직 (loader.py)

```python
def clean_journal(df):
    df = df[df['구분'].isin(['차변','대변'])].copy()
    df['전표일자'] = pd.to_datetime(df['전표일자'], errors='coerce')
    df = df[df['전표일자'].notna()]
    for col in ['차변','대변']:
        df[col] = pd.to_numeric(df[col].astype(str).str.replace(',',''), errors='coerce').fillna(0)
    df['계정그룹'] = df['계정코드'].astype(str).str[:1]
    return df
```

## 14. 원판관리 페이지 (✅ 완료 — pages/4_원판관리.py)

**업로드 방식 (8_데이터허브설정.py 탭5):**
```
파일 1: 원판구매내역_Yr{YYYY}-{MM}-{DD}.xlsx
파일 2: {YYYY}_{MM}_원판수불부_곡성.xlsx
→ 파일명에서 연월 자동 추출 (정규식 {YYYY}-{MM} 또는 Yr{YYYY}-{MM})
→ 시트탭명 = 거래처명 (KCC글라스/LX글라스/한유에스앤지/한성유엔씨)
→ 원산지 '원산지' 문자열 포함 행 필터링
→ 파일 기재 단가 vs 직접계산 단가 ±5원 초과 시 오기여부=true
```

**단가 계산 공식:**
```python
면적_m2 = (가로_m) × (세로_m) × 매수   # 1000 초과면 mm으로 간주 → /1000
원_m2   = 금액 / 면적_m2
원_평   = round(원_m2 / 10.764)
```

**화면 구성:**
```
[상단] 수불 집계 카드: 당기입고 / 당기사용 / 기초재고 / 기말재고 (매수+금액)
[중단] 단가 비교표: 두께×거래처×원산지 필터, 원/㎡·원/평 동시 표시
       두께별 평균단가 바차트 (거래처별 그룹)
[하단] 연간 탭: 월별 당기입고/사용 금액 바차트 + 집계 테이블
```

**Supabase 테이블 (최초 1회 SQL 실행 필요 — 8_데이터허브설정 탭5 참조):**
- `raw_material_price`: 품목별 단가 이력
- `raw_material_summary`: 월별 수불 집계
